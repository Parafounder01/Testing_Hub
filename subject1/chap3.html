DERIVATION SUMS (Step-by-Step)

1. Small-Signal Gain for BJT at Microwave

Steps:

  1. S₂₁ = forward transmission coefficient
  2. |S₂₁|² = power gain (linear)
  3. Gain(dB) = 20log₁₀|S₂₁|
  4. For unmatched: G = |S₂₁|²/(1-|S₁₁|²)

Formula: Gain(dB) = 20log₁₀|S₂₁|

2. Stability Factor K

Steps:

  1. K = (1-|S₁₁|²-|S₂₂|²-|Δ|²)/(2|S₁₂S₂₁|)
  2. Δ = S₁₁S₂₂ - S₁₂S₂₁
  3. K > 1 and |Δ| < 1 → Unconditionally stable
  4. K < 1 → Potentially unstable

Formula: K = (1-|S₁₁|²-|S₂₂|²-|Δ|²)/(2|S₁₂S₂₁|)

3. Output Power for Class A Amplifier

Steps:

  1. P_out = (V_Cc - V_sat)²/(2R_L)
  2. Collector efficiency: η = P_out/P_DC
  3. P_DC = V_Cc × I_Cq
  4. Max efficiency = 50% for Class A

Formula: P_out = (V_cc - V_sat)²/(2R_L), η = P_out/(V_cc×I_cq)

4. Transit Time Effects in Gunn Diodes

Steps:

  1. Gunn effect: transferred electron effect
  2. Negative differential resistance (NDR)
  3. Oscillation frequency: f = v_d/(2L) for domain
  4. v_d = drift velocity, L = active region length

Formula: f_osc = v_d/(2L)

5. Two-Port Power Gain

Steps:

  1. Transducer gain: G_T = P_L/(available from source)
  2. Available power: P_avs = |V_s|²/(8R_s)
  3. G_T = (|S₂₁|²(1-|Γ_s|²)(1-|Γ_L|²))/(|(1-S₁₁Γ_s)(1-S₂₂Γ_L) - S₁₂S₂₁Γ_sΓ_L|²)
  4. For matched: Γ_s = Γ_L = 0 → G_T = |S₂₁|²

Formula: G_T = |S₂₁|² for matched input/output

NUMERICAL PROBLEMS (Solved)

Problem 1: Calculate gain in dB given |S₂₁| = 2.5

Given: |S₂₁| = 2.5

Solution: Gain(dB) = 20log₁₀(2.5) = 20×0.3979 = 7.96 dB

Answer: 7.96 dB

Problem 2: Find stability factor K for S-params: S₁₁=0.5∠0°, S₂₂=0.4∠0°, S₁₂=0.1∠0°, S₂₁=2.5∠0°

Given: Calculate Δ = S₁₁S₂₂ - S₁₂S₂₁ = 0.2 - 0.25 = -0.05

Solution: K = (1-0.25-0.16-0.0025)/(2×0.25) = 0.5875/0.5 = 1.175

Answer: K = 1.175 > 1, Stable
