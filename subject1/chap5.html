Chapter 5: Microwave Integrated Circuits

Microwave Integrated Circuits (MIC) put components on a single substrate. Microstrip lines are like tiny transmission lines on PCB. They're used in cell phones and WiFi devices.

DERIVATION SUMS (Step-by-Step)

1. Microstrip Characteristic Impedance

Steps:

  1. For W/h > 1: Z₀ = (87/√(εᵣ+1.41))ln(5.98h/W)
  2. For W/h < 1: Z₀ = (60/√(εᵣ))ln(8h/W + W/4h)
  3. εᵣ = (εᵣ+1)/2 + (εᵣ-1)/2×(1+12h/W)^(-1/2)
  4. Effective dielectric constant

Formula: Z₀ = (87/√(εᵣ+1.41))ln(5.98h/W) for W/h > 1

2. Effective Dielectric Constant

Steps:

  1. ε_eff = (εᵣ+1)/2 + (εᵣ-1)/2×(1+12h/W)^(-1/2)
  2. For wide strips: ε_eff ≈ (εᵣ+1)/2
  3. Phase velocity: vp = c/√ε_eff
  4. Wavelength: λg = λ₀/√ε_eff

Formula: ε_eff = (εᵣ+1)/2 + (εᵣ-1)/2×(1+12h/W)^(-1/2)

3. Attenuation in Microstrip

Steps:

  1. Conductor loss: α_c = Rs/(Z₀W) nepers/m
  2. Rs = √(πfμ/σ) = surface resistance
  3. Dielectric loss: α_d = (π/λ)×(εᵣ/ε_eff)×(ε_eff-1)/(εᵣ-1)×tanδ
  4. Total: α = α_c + α_d

Formula: α_c = Rs/(Z₀W) Np/m, Rs = √(πfμ/σ)

4. Coupled Line Even/Odd Mode

Steps:

  1. Even mode: Z₀e = Z₀(1+C) (higher impedance)
  2. Odd mode: Z₀o = Z₀(1-C) (lower impedance)
  3. Coupling: C = (Z₀e-Z₀o)/(Z₀e+Z₀o)
  4. For directional coupler: C = 10^(-C_dB/20)

Formula: C = (Z₀e-Z₀o)/(Z₀e+Z₀o)

5. Butterworth Filter Design

Steps:

  1. Low-pass prototype: |H(jω)|² = 1/(1+ω²ⁿ)
  2. For n=3: poles at 60° intervals
  3. g₀=g₄=1, g₁=g₃=2sin(π/2n), g₂=2sin(π/2n) for n=3
  4. Transform to bandpass/bandstop

Formula: |H(jω)|² = 1/(1+(ω/ωc)²ⁿ) for Butterworth

NUMERICAL PROBLEMS (Solved)

Problem 1: Design 50Ω microstrip on FR4 (εᵣ=4.4, h=1.6mm), find W

Given: For Z₀=50Ω, use Z₀ ≈ 60/√εᵣ × ln(8h/W) → W/h ≈ 2

Solution: W = 2×1.6 = 3.2 mm (approximate)

Answer: W ≈ 3.2 mm

Problem 2: Calculate guide wavelength at 2.4 GHz on Rogers RO4350 (εᵣ=3.48)

Given: ε_eff ≈ (3.48+1)/2 = 2.24, λg = λ₀/√ε_eff = 12.5/1.5 = 8.33 cm

Solution: λg = 8.33 cm at 2.4 GHz

Answer: 8.33 cm
