Chapter 4: Microwave Measurements & Propagation

Measuring microwaves needs special tools like network analyzers. Smith chart is a graphical tool that helps match impedances. Link budget calculates how much power reaches the receiver.

DERIVATION SUMS (Step-by-Step)

1. Reflection Coefficient Γ

Steps:

  1. Γ = (ZL-Z₀)/(ZL+Z₀)
  2. For open circuit: ZL=∞ → Γ=+1
  3. For short circuit: ZL=0 → Γ=-1
  4. For matched load: ZL=Z₀ → Γ=0

Formula: Γ = (ZL-Z₀)/(ZL+Z₀) = |Γ|e^(jθ)

2. Transmission Coefficient T

Steps:

  1. T = VL/VS = 2ZL/(ZL+Z₀) for voltage
  2. Power transmission: τ = PL/Pin = 1-|Γ|²
  3. For lossless: |T|² = 1-|Γ|²
  4. Insertion loss = -10log₁₀|T|²

Formula: T = 2ZL/(ZL+Z₀), τ = 1-|Γ|²

3. Smith Chart Transformations

Steps:

  1. Impedance to admittance: rotate 180°
  2. z = Z/Z₀ = (1+Γ)/(1-Γ)
  3. y = Y/Y₀ = (1-Γ)/(1+Γ)
  4. Adding reactance: move along constant |Γ| circle

Formula: z = (1+Γ)/(1-Γ), y = 1/z

4. Friis Link Budget Equation

Steps:

  1. P_r = P_t × G_t × G_r × (λ/(4πd))²
  2. Path loss: L_fs = (4πd/λ)²
  3. In dB: P_r(dBm) = P_t(dBm) + G_t(dB) + G_r(dB) - L_fs(dB)
  4. L_fs(dB) = 20log₁₀(d) + 20log₁₀(f) - 147.55

Formula: P_r = P_t G_t G_r (λ/4πd)²

5. Return Loss

Steps:

  1. RL = -20log₁₀|Γ| dB
  2. For VSWR = 2: |Γ| = (2-1)/(2+1) = 1/3
  3. RL = -20log₁₀(1/3) = 9.54 dB
  4. Better match → higher return loss

Formula: RL(dB) = -20log₁₀|Γ| = 20log₁₀(VSWR+1)/(VSWR-1)

NUMERICAL PROBLEMS (Solved)

Problem 1: Use Smith chart to match 50Ω to 75Ω at 1 GHz

Given: Z_in = 50Ω, Z_L = 75Ω, normalize: z_L = 75/50 = 1.5

Solution: Use stub matching: add inductive stub at d/λg = 0.125 from load

Answer: Stub position: 0.125λg from load

Problem 2: Calculate received power for P_t=1W, G_t=20 dBi, G_r=20 dBi, d=10 km, f=2.4 GHz

Given: L_fs = 20log₁₀(10⁴) + 20log₁₀(2.4×10⁹) - 147.55 = 40 + 67.6 - 147.55 = -39.95 dB

Solution: P_r = 30 + 20 + 20 - 39.95 = 30.05 dBm = 1 W

Answer: P_r = 1 W (30.05 dBm)
