Chapter 1: Waveguide Theory & Modes

Hello student! Today we learn about Waveguides. A waveguide is like a pipe that carries microwaves from one place to another. Unlike regular wires, waveguides use the air inside metal pipes to guide electromagnetic waves.

DERIVATION SUMS (Step-by-Step)

1. Telegrapher's Equations from Maxwell

Steps:

  1. Start with Maxwell's equations in phasor form
  2. Apply to coaxial line geometry
  3. Derive: ∂V/∂z = -(R+jωL)I
  4. Derive: ∂I/∂z = -(G+jωC)V
  5. Result: Wave equation ∂²V/∂z² = γ²V

Formula: γ = √((R+jωL)(G+jωC)) [propagation constant]

2. Characteristic Impedance Z₀

Steps:

  1. For lossless line: R=0, G=0
  2. γ = jω√(LC)
  3. Z₀ = V+/I+ = √(L/C)
  4. For coaxial: Z₀ = (1/2π)√(μ/ε) × ln(b/a)

Formula: Z₀ = √(L/C) = (60/√εᵣ) × ln(b/a) for coaxial

3. Cutoff Frequency for Rectangular Waveguide

Steps:

  1. For TE₁₀ mode in rectangular waveguide
  2. Cutoff wavelength λc = 2a (a = wider dimension)
  3. fc = c/λc = c/(2a)
  4. Higher modes have higher cutoff frequencies

Formula: fc = c/(2a) for TE₁₀ mode

4. Phase and Group Velocity

Steps:

  1. Phase velocity: vp = ω/β
  2. Group velocity: vg = dω/dβ
  3. For waveguides: vp = c/√(1-(fc/f)²)
  4. vg = c×√(1-(fc/f)²)
  5. Note: vp × vg = c² (product constant)

Formula: vp = fλg = ω/β, vg = dω/dβ = c√(1-(fc/f)²)

5. Waveguide Impedance

Steps:

  1. For TE modes: Z_w = β/(ωε) = η/√(1-(fc/f)²)
  2. For TM modes: Z_w = η×√(1-(fc/f)²)
  3. η = √(μ/ε) = intrinsic impedance
  4. At f >> fc: Z_w ≈ η

Formula: Z_TE = η/√(1-(fc/f)²), Z_TM = η√(1-(fc/f)²)

NUMERICAL PROBLEMS (Solved)

Problem 1: Find cutoff frequency for WR-90 waveguide (a=0.9", b=0.4")

Given: a = 0.9 inch = 0.02286 m, c = 3×10⁸ m/s

Solution: fc = c/(2a) = 3×10⁸/(2×0.02286) = 6.56 GHz

Answer: 6.56 GHz

Problem 2: Calculate guide wavelength λg at f=10 GHz for a=2.286 cm waveguide

Given: a = 2.286 cm, f = 10 GHz, fc = c/(2a) = 6.56 GHz

Solution: λg = λ/√(1-(fc/f)²) = (3×10⁸/10⁹)/√(1-(6.56/10)²) = 3.6 cm

Answer: 3.6 cm

Problem 3: Find S-parameters for a lossless transmission line of length l

Given: Characteristic impedance Z₀, electrical length θ=βl

Solution: S₁₁ = S₂₂ = 0 (matched), S₂₁ = S₁₂ = e^(-jθ) (transmission)

Answer: S-matrix = [[0, e^(-jθ)], [e^(-jθ), 0]]
