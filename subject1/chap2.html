Chapter 2: Microwave Passive Components

Microwave passive components help split, combine, or control microwave signals without adding power. They are like water pipes with valves and splitters.

DERIVATION SUMS (Step-by-Step)

1. S-Matrix for Directional Coupler

Steps:

  1. 4-port network: input, output, coupled, isolated
  2. Forward coupling: S₃₁ = C (coupling factor)
  3. Through path: S₂₁ = T (transmission)
  4. Isolation: S₄₁ = 0 (ideal)
  5. Reciprocity: Sᵢⱼ = Sⱼᵢ

Formula: |S₃₁|² = coupling, |S₂₁|² = 1-coupling (ideal)

2. VSWR from Reflection Coefficient

Steps:

  1. Reflection coefficient: Γ = (ZL-Z₀)/(ZL+Z₀)
  2. VSWR = (1+|Γ|)/(1-|Γ|)
  3. Return loss = -20log|Γ| dB
  4. For perfect match: Γ=0, VSWR=1

Formula: VSWR = (1+|Γ|)/(1-|Γ|), RL = -20log₁₀|Γ| dB

3. Insertion Loss

Steps:

  1. IL = 10log₁₀(P_in/P_out) dB
  2. IL = 10log₁₀(1/|S₂₁|²) dB
  3. For coupler: IL = -10log₁₀(1-|C|²)
  4. Mismatch loss also contributes

Formula: IL = 10log₁₀(1/|S₂₁|²) dB

4. Phase Shifter Group Delay

Steps:

  1. Phase shift φ = βl = (2π/λg)l
  2. Group delay τg = dφ/dω = l/vg
  3. For f=10 GHz, l=λg/4: φ = 90°
  4. Electrical length: θ = βl radians

Formula: τg = dφ/dω = l/vg = l√(1-(fc/f)²)/c

5. Wilkinson Power Divider

Steps:

  1. 2-way divider with isolation resistors
  2. Input matched: Z_in = Z₀
  3. Output ports isolated: S₃₂ = 0
  4. Power division: P₂ = P₃ = P₁/2

Formula: Z_in = Z₀, P_out = P_in/2 (3 dB split)

NUMERICAL PROBLEMS (Solved)

Problem 1: Design 10 dB directional coupler (find coupling, isolation)

Given: C = 10 dB = 0.1 (linear), P_coupled/P_in = 0.1

Solution: Coupling = 10log₁₀(0.1) = -10 dB, Through = 10log₁₀(0.9) = -0.46 dB

Answer: Coupling = -10 dB, Through = -0.46 dB

Problem 2: Find VSWR given return loss = 20 dB

Given: RL = 20 dB → |Γ| = 10^(-20/20) = 0.1

Solution: VSWR = (1+0.1)/(1-0.1) = 1.22

Answer: VSWR = 1.22:1
