Chapter 4: Pulse & Wave-Shaping Circuits

Pulse circuits shape voltage waveforms. 555 timer is like a kitchen timer that beeps periodically. Clipper cuts off voltage above/below limit. Schmitt trigger has two thresholds to avoid flicker.

DERIVATION SUMS (Step-by-Step)

1. 555 Astable Frequency

Steps:

  1. f = 1.44/((R₁+2R₂)C) for 50% duty cycle
  2. t_high = 0.693(R₁+R₂)C
  3. t_low = 0.693R₂C
  4. Duty cycle = (R₁+R₂)/(R₁+2R₂)
  5. For 50%: R₁ << R₂

Formula: f = 1.44/((R₁+2R₂)C)

2. 555 Monostable Pulse Width

Steps:

  1. T = 1.1×R×C (one-shot pulse)
  2. Trigger pulse < T (shorter than output)
  3. Pulse width independent of trigger width
  4. Retriggerable: trigger during pulse extends T
  5. Non-retriggerable: ignores subsequent triggers

Formula: T = 1.1RC

3. RC Time Constant

Steps:

  1. τ = RC (charges to 63.2% in time τ)
  2. V_c(t) = V_cc(1-e^(-t/τ)) for charging
  3. V_c(t) = V_0e^(-t/τ) for discharging
  4. After 5τ: capacitor ≈ fully charged/discharged

Formula: τ = RC, V_c(t) = V_cc(1-e^(-t/τ))

4. Clipping Level (Diode Clipper)

Steps:

  1. V_out = V_in for V_in < V_clip
  2. V_out = V_clip for V_in ≥ V_clip (positive clipper)
  3. With Zener: V_clip = V_Z (reverse breakdown)
  4. Bidirectional: two Zeners back-to-back

Formula: V_out = min(V_in, V_clip)

5. Schmitt Trigger Thresholds

Steps:

  1. V_T+ = βV_cc (upper threshold)
  2. V_T- = β'V_cc (lower threshold)
  3. β = R₁/(R₁+R₂) (voltage divider)
  4. Hysteresis = V_T+ - V_T-
  5. Noise immunity ∝ hysteresis width

Formula: V_T+ = (R₁/(R₁+R₂))V_cc

NUMERICAL PROBLEMS (Solved)

Problem 1: Design 555 astable: f=1kHz, duty cycle=60% → find R₁, R₂, C

Given: f = 1.44/((R₁+2R₂)C), duty = (R₁+R₂)/(R₁+2R₂) = 0.6

Solution: Choose C=0.1μF. R₁+2R₂ = 1.44/(1000×0.1μ) = 14.4kΩ. R₁+R₂ = 0.6×14.4k = 8.64kΩ. Solve: R₂ = 14.4k-8.64k = 5.76kΩ, R₁ = 8.64k-5.76k = 2.88kΩ. Choose R₁=2.7k, R₂=5.6k standard values

Answer: R₁≈2.7kΩ, R₂≈5.6kΩ, C=0.1μF

Problem 2: Calculate monostable pulse: R=10kΩ, C=0.1μF

Given: T = 1.1RC

Solution: T = 1.1×10k×0.1μ = 1.1×10⁴×10⁻⁷ = 1.1×10⁻³ = 1.1ms

Answer: T = 1.1ms
