Chapter 1: Semiconductor Devices & Physics

Hello student! Today we learn about semiconductor devices. Diode is like a one-way street for electricity. BJT uses small base current to control large collector current. MOSFET uses voltage to control current.

DERIVATION SUMS (Step-by-Step)

1. Diode Equation

Steps:

  1. I_D = I_S(e^(V_D/nV_T)-1)
  2. V_T = kT/q = 26mV at 300K
  3. I_S = reverse saturation current (very small)
  4. For V_D >> V_T: I_D ≈ I_S e^(V_D/nV_T)
  5. For V_D << -V_T: I_D ≈ -I_S (reverse bias)

Formula: I_D = I_S(e^(V_D/nV_T)-1)

2. BJT Current Gain

Steps:

  1. β = I_C/I_B (DC current gain)
  2. α = I_C/I_E = β/(β+1)
  3. I_C = βI_B + (β+1)I_CBO
  4. For large β: I_C ≈ βI_B
  5. I_E = I_B + I_C = (β+1)I_B

Formula: β = I_C/I_B, α = β/(β+1)

3. MOSFET Drain Current (Saturation)

Steps:

  1. I_D = (μ_nC_ox/2)(W/L)(V_GS-V_th)²
  2. μ_n = electron mobility
  3. C_ox = ε_ox/t_ox (oxide capacitance)
  4. V_DS ≥ V_GS - V_th (saturation condition)
  5. For λ ≠ 0: I_D = I_D0(1+λV_DS) (channel length modulation)

Formula: I_D = (μ_nC_ox/2)(W/L)(V_GS-V_th)²

4. JFET Pinch-off Voltage

Steps:

  1. V_p = (eN_d/2ε_s)×a² (for abrupt junction)
  2. a = channel radius
  3. N_d = donor concentration (n-channel)
  4. I_D = I_DSS(1 - V_GS/V_p)² (Shockley equation)
  5. Pinch-off: V_GS = V_p → I_D = 0

Formula: V_p = (eN_d/2ε_s)×a²

5. SCR Turn-on Condition

Steps:

  1. Latching current I_L > holding current I_H
  2. Forward bias: anode-to-cathode voltage > 0.7V
  3. Gate current triggers turn-on
  4. Once latched, gate loses control
  5. Turn-off: anode current < I_H

Formula: I_L > I_H for latching

NUMERICAL PROBLEMS (Solved)

Problem 1: Calculate diode current: I_S=10^-12A, V_D=0.7V, T=300K

Given: V_T = kT/q = 26mV = 0.026V

Solution: I_D = 10^-12 × (e^(0.7/0.026) - 1) = 10^-12 × (e^26.92 - 1) ≈ 10^-12 × 4.36×10^11 = 0.436A

Answer: I_D ≈ 0.436A

Problem 2: Find BJT currents: I_B=20μA, β=100

Given: β = I_C/I_B = 100

Solution: I_C = β×I_B = 100×20μA = 2mA. I_E = I_B + I_C = 20μA + 2mA = 2.02mA. α = β/(β+1) = 100/101 = 0.99

Answer: I_C=2mA, I_E=2.02mA, α=0.99

Problem 3: Design MOSFET amplifier: V_GS=3V, V_th=1V, W/L=10, μ_nC_ox=100μA/V²

Given: I_D = (μ_nC_ox/2)(W/L)(V_GS-V_th)²

Solution: I_D = (100μ/2)×10×(3-1)² = 50μ×10×4 = 2000μA = 2mA

Answer: I_D = 2mA
