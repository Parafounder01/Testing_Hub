Chapter 5: Applications & Power Circuits

Power circuits supply clean DC voltage. Zener diode limits voltage like a pressure relief valve. SMPS uses switches for high efficiency. Rectifiers convert AC to DC.

DERIVATION SUMS (Step-by-Step)

1. Zener Regulator Design

Steps:

  1. V_out = V_Z (Zener voltage)
  2. I_Z(min) < I_Z < I_Z(max) for regulation
  3. R_S = (V_in - V_Z)/(I_L + I_Z)
  4. P_Z = V_Z × I_Z(max) (Zener power rating)
  5. Line regulation = ΔV_out/ΔV_in

Formula: R_S = (V_in - V_out)/(I_L + I_Z)

2. SMPS Buck Converter

Steps:

  1. V_out = D×V_in (duty cycle D)
  2. D = t_on/T (switch on time/total period)
  3. ΔI_L = (V_in-V_out)×D×T/L
  4. Output ripple: ΔV_out ≈ ΔI_L×ESR
  5. Efficiency η > 85% (much better than linear)

Formula: V_out = D×V_in, D = t_on/T

3. Rectifier Ripple Voltage

Steps:

  1. Full-wave bridge: V_ripple = I_L/(2fC)
  2. Half-wave: V_ripple = I_L/(fC)
  3. V_DC ≈ V_peak - V_ripple/2
  4. Capacitor stores energy between peaks
  5. Larger C → smaller ripple

Formula: V_ripple = I_L/(2fC) [full-wave], I_L/(fC) [half-wave]

4. Regulator Efficiency

Steps:

  1. η = P_out/P_in = (V_out×I_L)/(V_in×I_in)
  2. Linear regulator: η = V_out/V_in (low, ~50%)
  3. SMPS: η > 85% (high efficiency)
  4. Power dissipation: P_D = (V_in-V_out)×I_L
  5. Heat sink needed for large P_D

Formula: η = V_out/V_in [linear], >85% [SMPS]

5. Current Limiting (Protection)

Steps:

  1. Sense resistor: V_sense = I_L×R_sense
  2. When V_sense > V_BE: transistor turns on
  3. Limits I_L(max) = V_BE/R_sense
  4. Typical V_BE = 0.7V for silicon
  5. Protects regulator from overcurrent

Formula: I_L(max) = V_BE/R_sense

NUMERICAL PROBLEMS (Solved)

Problem 1: Design Zener regulator: V_out=5V, I_L=50mA, V_in=12V → find R_S, P_Z

Given: Choose I_Z = 10mA. R_S = (12-5)/(50m+10m) = 7/0.06 = 116.7Ω. Choose 120Ω standard. P_Z(max) = 5V×10mA = 50mW. Choose 0.5W Zener for safety

Solution: R_S = 120Ω, P_Z = 0.5W Zener

Answer: R_S≈120Ω, P_Z=0.5W

Problem 2: Calculate buck converter: V_in=24V, V_out=12V → find D

Given: V_out = D×V_in

Solution: D = V_out/V_in = 12/24 = 0.5 = 50% duty cycle

Answer: D = 0.5 (50%)

Problem 3: Design bridge rectifier: V_out=12V DC, I_L=1A → find C for V_ripple<0.1V

Given: Full-wave: V_ripple = I_L/(2fC), f=50Hz

Solution: C = I_L/(2f×V_ripple) = 1/(2×50×0.1) = 1/10 = 0.1F = 100,000μF. Choose 100,000μF capacitor

Answer: C = 100,000μF
