Chapter 3: IC Design & Fabrication

IC fabrication is like building tiny houses on silicon. Lithography transfers patterns using light. Oxidation grows controlled 'rust' that becomes insulator. Diffusion implants impurities.

DERIVATION SUMS (Step-by-Step)

1. Lithography Resolution (Rayleigh Criterion)

Steps:

  1. R = k₁λ/NA (minimum feature size)
  2. λ = wavelength of light
  3. NA = n sinθ (numerical aperture)
  4. k₁ = process-dependent constant (~0.5)
  5. Smaller λ or larger NA → finer features

Formula: R = k₁λ/NA

2. Oxidation Rate (Deal-Grove Model)

Steps:

  1. Parabolic: x² = B(t+τ) for thick oxide
  2. Linear: x = B/A·t for thin oxide
  3. B = parabolic rate constant
  4. A = linear rate constant
  5. τ = time shift (initial rapid growth)

Formula: x² = B(t+τ) [parabolic], x = (B/A)t [linear]

3. Diffusion Profile (Constant Source)

Steps:

  1. C(x,t) = C₀erfc(x/√(4Dt))
  2. C₀ = surface concentration
  3. D = diffusion coefficient
  4. erfc = complementary error function
  5. J = -D∂C/∂x (Fick's first law)

Formula: C(x,t) = C₀erfc(x/√(4Dt))

4. CMOS Inverter Delay

Steps:

  1. τ = (C_L×V_DD)/I_D(sat) (switching delay)
  2. I_D(sat) = (μ_nC_ox/2)(W/L)(V_DD-V_th)²
  3. Propogation delay: t_pd = (t_pHL + t_pLH)/2
  4. For equal rise/fall: W_p/W_n = μ_n/μ_p
  5. Power: P = C_L×V_DD²×f (dynamic)

Formula: τ = (C_L×V_DD)/I_D(sat)

5. Layout Area for Transistor

Steps:

  1. A = (W+L)×(feature size)²
  2. Feature size = minimum gate length L_min
  3. Contact area adds overhead
  4. Standard cell: fixed height, variable width
  5. Area ∝ W×L for simple transistor

Formula: A ≈ (W+L)×L_min²

NUMERICAL PROBLEMS (Solved)

Problem 1: Calculate oxide thickness: t=30min, B=0.1μm²/hr → find x

Given: x² = B(t+τ), assume τ≈0

Solution: t = 30min = 0.5hr. x = √(0.1×0.5) = √0.05 = 0.2236μm = 223.6nm

Answer: x ≈ 224nm

Problem 2: Find diffusion depth: t=10min, D=10^-12 m²/s

Given: C(x,t) = C₀erfc(x/√(4Dt))

Solution: x = √(4Dt) for C/C₀ = erfc(1) ≈ 0.157. t = 600s. x = √(4×10^-12×600) = √2.4×10^-9 = 4.9×10^-5 m = 49μm

Answer: x ≈ 49μm at 1% of surface
