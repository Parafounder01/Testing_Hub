DERIVATION SUMS (Step-by-Step)

1. Boolean Algebra Laws

Steps:

  1. Commutative: A+B = B+A, A·B = B·A
  2. Associative: (A+B)+C = A+(B+C)
  3. Distributive: A+(B·C) = (A+B)·(A+C)
  4. De Morgan: (A+B)' = A'·B', (A·B)' = A'+B'
  5. Absorption: A+(A·B) = A, A·(A+B) = A

Formula: De Morgan: (A+B)' = A'·B'

2. Karnaugh Map Simplification

Steps:

  1. F = Σ(0,2,4,6) for 3-variable K-map
  2. Group adjacent 1s in powers of 2
  3. Group 1: m0+m2 = A'B'C'+A'BC' = A'C'
  4. Group 2: m4+m6 = AB'C'+ABC' = BC'
  5. F = A'C' + BC' = C'(A'+B)

Formula: F = Σ(minterms) → group → simplified expression

3. Flip-Flop Characteristic Equations

Steps:

  1. D-FF: Q⁺ = D (next state = D input)
  2. JK-FF: Q⁺ = JQ' + K'Q
  3. SR-FF: Q⁺ = S + R'Q (with SR=0 constraint)
  4. T-FF: Q⁺ = T⊕Q = TQ' + T'Q
  5. Clocked: changes only on clock edge

Formula: JK-FF: Q⁺ = JQ' + K'Q

4. Counter Modulus

Steps:

  1. n-bit counter: modulus M = 2ⁿ
  2. MOD-16 counter: n=4, M=16 states
  3. Frequency division: f_out = f_clk/2ⁿ
  4. Decade counter: M=10 (uses 4 bits)
  5. Design: connect Q to reset when count=9

Formula: M = 2ⁿ for n-bit counter

5. Setup & Hold Time

Steps:

  1. t_setup: data must be stable BEFORE clock edge
  2. t_hold: data must be stable AFTER clock edge
  3. Clock period: T_clk > t_cq(max) + t_setup + t_logic
  4. Max frequency: f_max = 1/T_clk
  5. Violation → metastability

Formula: T_clk > t_cq(max) + t_logic + t_setup

NUMERICAL PROBLEMS (Solved)

Problem 1: Simplify Boolean: F = AB + AB' + A'B using Boolean algebra

Given: Use Boolean laws

Solution: F = A(B+B') + A'B = A·1 + A'B = A + A'B = A + B (by consensus)

Answer: F = A + B

Problem 2: Design 3-bit counter using JK-FF (draw circuit, timing)

Given: MOD-8 counter: count 0 to 7

Solution: Connect J=K=1 for all 3 FF. Q0 toggles every clock. Q1 toggles on Q0' edge. Q2 toggles on Q1' edge. Timing: 000→001→010→011→100→101→110→111→000

Answer: 3-bit ripple counter, J=K=1 for all
