Chapter 5: Op-Amps, Power Amps & Supplies

Op-amps are versatile ICs that perform mathematical operations. Inverting amplifier flips signal, non-inverting keeps phase same. Class B power amps use two transistors for better efficiency.

DERIVATION SUMS (Step-by-Step)

1. Inverting Amplifier Gain

Steps:

  1. Virtual ground at inverting terminal (V- ≈ 0)
  2. I_in = (V_in - 0)/R1 = V_in/R1
  3. I_f = (0 - V_out)/Rf = -V_out/Rf
  4. I_in = I_f (no current into op-amp)
  5. V_out/V_in = -Rf/R1

Formula: Av = V_out/V_in = -Rf/R1

2. Non-Inverting Amplifier Gain

Steps:

  1. Voltage at inverting terminal: V- = V_out×R1/(R1+Rf)
  2. V+ = V_in (virtual short: V+ = V-)
  3. V_in = V_out×R1/(R1+Rf)
  4. V_out/V_in = (R1+Rf)/R1 = 1 + Rf/R1

Formula: Av = 1 + Rf/R1

3. Class A Maximum Efficiency

Steps:

  1. P_out = (Vcc/√2)²/R_L = Vcc²/(2R_L) (max before clipping)
  2. P_DC = Vcc × I_cq = Vcc × (Vcc/2R_L) = Vcc²/(2R_L)
  3. η_max = P_out/P_DC = 50%
  4. Practical: η ≈ 25-40% due to V_sat, Re

Formula: η_max = 50% for Class A

4. Class B Maximum Efficiency

Steps:

  1. Each transistor conducts 180° (half cycle)
  2. P_out = (Vcc/√2)²/(2R_L) = Vcc²/(4R_L)
  3. P_DC = 2×Vcc×(Vcc/πR_L) = 2Vcc²/(πR_L)
  4. η_max = P_out/P_DC = π/4 ≈ 78.5%

Formula: η_max = π/4 ≈ 78.5% for Class B

5. Voltage Regulator Output (LM317)

Steps:

  1. V_out = V_ref×(1 + R2/R1) + I_adj×R2
  2. V_ref ≈ 1.25V for LM317
  3. I_adj ≈ 50μA (neglect for approximation)
  4. V_out ≈ 1.25×(1 + R2/R1)

Formula: V_out = 1.25×(1 + R2/R1) for LM317

NUMERICAL PROBLEMS (Solved)

Problem 1: Design inverting amplifier: Av=-10, Rin=1kΩ → find Rf

Given: Av = -Rf/R1 = -10, R1 = 1kΩ

Solution: Rf = -Av×R1 = 10×1k = 10kΩ

Answer: Rf = 10kΩ

Problem 2: Calculate Class B power: Vcc=15V, RL=8Ω → find P_out, P_DC, η

Given: V_peak = Vcc = 15V (ideal)

Solution: P_out = V_peak²/(2R_L) = 15²/(2×8) = 225/16 = 14.06W. P_DC = 2×15×(15/π×8) = 450/(25.13) = 17.9W. η = 14.06/17.9 = 78.5%

Answer: P_out=14.06W, P_DC=17.9W, η=78.5%
