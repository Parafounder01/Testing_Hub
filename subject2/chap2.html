Chapter 2: Amplifiers (Single & Multistage)

Amplifiers make signals bigger. CE (Common Emitter) gives high gain but inverts signal. CC (Common Collector) is emitter follower with gain≈1 but high input impedance.

  • CE/CB/CC Configurations

  • Input/Output Impedance
  • Multistage Gain

DERIVATION SUMS (Step-by-Step)

1. Voltage Gain Av for CE Amplifier

Steps:

  1. Av = Vce/Vbe = -Ic×RL'/Ib×hie
  2. Ic = hfe×Ib (approximate)
  3. Av = -hfe×RL'/hie
  4. With load: RL' = Rc||RL
  5. Exact: Av = -hfe×RL'/(hie + (hfe×RL'×hre))

Formula: Av = -hfe×RL'/hie (approx), Av = -hfe×RL'/(hie + hfe×RL'×hre) (exact)

2. Input Impedance Ri for CE

Steps:

  1. Ri = Vbe/Ib = hie + (1+hfe)×Re
  2. With feedback: Ri = hie + hfe×hre×RL'
  3. Source loading: Rin = Ri || Rth
  4. For CC: Ric = hie + (1+hfe)×Re (high input Z)

Formula: Ri = hie + (1+hfe)×Re

3. Current Gain Ai for CE

Steps:

  1. Ai = Ic/Ib = hfe/(1 + hoe×RL)
  2. If hoe×RL << 1: Ai ≈ hfe
  3. With source: Ais = Ai × Zin/(Zin + Rs)
  4. For CC: Aic = -(1+hfe) ≈ -β

Formula: Ai = hfe/(1 + hoe×RL)

4. Output Impedance Ro for CE

Steps:

  1. Ro = 1/hoe || Rc (approximate)
  2. Exact: Ro = 1/(hoe - hfe×hre/(hie+Rs))
  3. With load: Rout = Ro || RL
  4. For CB: Rob = 1/hoe (very high)

Formula: Ro ≈ 1/hoe || Rc

5. Overall Gain for 2-Stage Amplifier

Steps:

  1. Av_total = Av1 × Av2 (if stages isolated)
  2. With loading: Av2_loaded = Av2 × Zin2/(Zin2 + Rout1)
  3. Overall current gain: Ai_total = Ai1 × Ai2
  4. Power gain: Ap = Av × Ai × cosφ

Formula: Av_total = Av1 × Av2 (ideal), Av_total = Av1 × (Av2×Zin2/(Zin2+Rout1)) (loaded)

NUMERICAL PROBLEMS (Solved)

Problem 1: Calculate Av, Ai, Ri, Ro for CE amplifier: hie=1.1k, hfe=50, hoe=25μS, Rc=2k, RL=2k, Rs=1k

Given: RL' = 2k||2k = 1kΩ, hoe×RL' = 25μ×1k = 0.025 << 1

Solution: Av = -50×1k/1.1k = -45.45. Ai = 50/(1+0.025) = 48.78. Ri = 1.1kΩ. Ro = 1/(25μ) || 2k = 40k || 2k = 1.9kΩ

Answer: Av=-45.45, Ai=48.78, Ri=1.1kΩ, Ro=1.9kΩ

Problem 2: Design 2-stage amplifier with total gain >1000, Av1=-50, find Av2 and Rc2

Given: hfe=50, hie=1.1k, Re=500Ω for stage 2

Solution: Av2_min = 1000/50 = 20. Av2 = -hfe×Rc2/hie → Rc2 = -Av2×hie/hfe = 20×1.1k/50 = 440Ω. Choose Rc2=470Ω

Answer: Av2=20+, Rc2≈470Ω
