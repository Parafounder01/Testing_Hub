Chapter 3: Feedback Amplifiers

Feedback means taking some output and feeding it back to input. Negative feedback reduces gain but improves stability, bandwidth, and reduces distortion.

LET'S REMEMBER:

  • Negative Feedback
  • Gain with Feedback Af
  • Input/Output Impedance with Feedback
  • Barkhausen Criterion

SIMPLE NOTES (Tanglish Supported):

  • Negative Feedback

DERIVATION SUMS (Step-by-Step)

1. Gain with Feedback Af

Steps:

  1. Af = Vo/Vs = A/(1 + Aβ)
  2. A = open-loop gain, β = feedback factor
  3. If Aβ >> 1: Af ≈ 1/β (desensitized)
  4. Desensitivity D = 1/(1+Aβ)

Formula: Af = A/(1 + Aβ)

2. Input Impedance with Feedback

Steps:

  1. Series feedback (voltage-series): Zif = Zi×(1+Aβ)
  2. Shunt feedback (current-shunt): Zif = Zi/(1+Aβ)
  3. For voltage-series: input Z increases with feedback
  4. For current-shunt: input Z decreases with feedback

Formula: Zif = Zi(1+Aβ) for series, Zif = Zi/(1+Aβ) for shunt

3. Output Impedance with Feedback

Steps:

  1. Voltage feedback (series-shunt): Zof = Zo/(1+Aβ)
  2. Current feedback (shunt-series): Zof = Zo×(1+Aβ)
  3. For voltage-series: output Z decreases (better regulation)
  4. For current-shunt: output Z increases

Formula: Zof = Zo/(1+Aβ) for voltage, Zof = Zo(1+Aβ) for current

4. Barkhausen Criterion for Oscillation

Steps:

  1. For oscillation: Aβ = -1 (magnitude 1, phase 180°)
  2. |Aβ| = 1 and ∠Aβ = 180° (or 360° total)
  3. Practical: |Aβ| > 1 at ∠Aβ = 180°
  4. Phase shift oscillator: 3 RC stages × 60° each = 180°

Formula: Aβ = -1 for oscillation (|Aβ|=1, ∠Aβ=180°)

5. Desensitivity D

Steps:

  1. D = (dA/A)/(dAf/Af) = 1/(1+Aβ)
  2. Larger Aβ = less gain variation with temperature
  3. Example: β=0.1, A=100 → D=1/(1+10)=0.0909
  4. Gain variation reduced by factor D

Formula: D = 1/(1+Aβ)

NUMERICAL PROBLEMS (Solved)

Problem 1: Calculate Af, Rif, Rof for voltage-series feedback: A=-100, β=0.1, Ri=1k, Ro=5k

Given: Aβ = -100×0.1 = -10

Solution: Af = -100/(1-10) = -100/-9 = 11.11. Rif = 1k×(1+10) = 11kΩ. Rof = 5k/|1-10| = 5k/9 = 555.6Ω

Answer: Af=11.11, Rif=11kΩ, Rof=555.6Ω

Problem 2: Find phase shift at f=10kHz for RC phase shift oscillator (3 stages)

Given: Each RC stage: φ = -arctan(2πfRC), R=10k, C=0.01μF

Solution: f₀=1/(2πRC√6) = 1/(2π×10k×0.01μ×2.45) = 650Hz. At 10kHz: φ per stage ≈ -arctan(10k/650) ≈ -86.3°. 3 stages = -258.9° ≈ -180° (oscillates)

Answer: Each stage ≈ -86.3°, Total ≈ -258.9° ≈ -180° (oscillates)
