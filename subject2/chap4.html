Chapter 4: Oscillators

Oscillators generate AC signals without input. RC phase shift uses resistors and capacitors. LC oscillators (Hartley/Colpitts) use inductors and capacitors for high frequency.

DERIVATION SUMS (Step-by-Step)

1. Frequency for RC Phase Shift Oscillator

Steps:

  1. 3-stage RC ladder network
  2. Each stage phase shift: φ = arctan(1/2πfRC)
  3. Total phase shift: 3φ = 180°
  4. f₀ = 1/(2πRC√6) for 3-stage
  5. Gain required: A > 29 (for sustained oscillation)

Formula: f₀ = 1/(2πRC√6)

2. Frequency for Hartley Oscillator

Steps:

  1. L_eq = L1 + L2 + 2M (with mutual inductance M)
  2. f₀ = 1/(2π√(L_eq×C))
  3. For no mutual coupling: f₀ = 1/(2π√((L1+L2)×C))
  4. Tapping point provides feedback

Formula: f₀ = 1/(2π√(L1+L2)C) (no M)

3. Frequency for Colpitts Oscillator

Steps:

  1. C_eq = (C1×C2)/(C1+C2) (series combination)
  2. f₀ = 1/(2π√(LC_eq))
  3. Voltage divider using C1, C2 provides feedback
  4. Gain required: A > C2/C1

Formula: f₀ = 1/(2π√(L×C1×C2/(C1+C2)))

4. Frequency for Wein Bridge Oscillator

Steps:

  1. Lead-lag network: f₀ = 1/(2πRC)
  2. Positive feedback through lead-lag network
  3. Negative feedback through Rf, Re
  4. Condition: Rf/Re = 2 for sustained oscillation

Formula: f₀ = 1/(2πRC), Rf = 2Re

5. Crystal Oscillator Q-Factor

Steps:

  1. Crystal equivalent: L, C1, R, C0 (motional params)
  2. Q = ωL/R = 1/(ωRC1) (very high: 10⁴-10⁶)
  3. f₀ = 1/(2π√LC1) (series resonance)
  4. Parallel resonance: fp = fs√(1+C1/C0)

Formula: Q = ωL/R, fs = 1/(2π√LC1)

NUMERICAL PROBLEMS (Solved)

Problem 1: Design RC phase shift oscillator for f₀=1kHz, R=10kΩ, find C

Given: f₀ = 1/(2πRC√6) = 1000 Hz

Solution: C = 1/(2π×10k×1000×2.45) = 1/(153.9×10⁶) = 6.5nF. Choose C=6.8nF standard

Answer: C ≈ 6.8nF

Problem 2: Calculate Hartley oscillator frequency: L1=1mH, L2=1mH, C=0.01μF (no mutual coupling)

Given: L_eq = L1+L2 = 2mH

Solution: f₀ = 1/(2π√(2m×0.01μ)) = 1/(2π√20n) = 1/(2π×4.47×10⁻⁵) = 3.56kHz

Answer: f₀ ≈ 3.56 kHz
