Chapter 1: Biasing Techniques

Hello student! Today we learn about biasing. Bias means giving transistor a 'job' - set its working point so it works properly. Like setting a fan speed before using it.

DERIVATION SUMS (Step-by-Step)

1. h-Parameters from Hybrid Model

Steps:

  1. V1 = h11 i1 + h12 V2
  2. i2 = h21 i1 + h22 V2
  3. h11 = (V1/i1) with V2=0 (input impedance)
  4. h21 = (i2/i1) with V2=0 (forward current gain)
  5. h12 = (V1/V2) with i1=0 (reverse voltage ratio)
  6. h22 = (i2/V2) with i1=0 (output admittance)

Formula: Vbe = hie Ib + hre Vce, Ic = hfe Ib + hoe Vce

2. Stability Factor S

Steps:

  1. S = dIc/dβ × (β+1)/Ic
  2. For voltage divider bias: S = (β+1)/(β+1 + Rth/Re)
  3. Lower S = better stability against β variation
  4. Fixed bias: S = β+1 (very poor stability)

Formula: S = (β+1)/(β+1 + Rth/Re) for voltage divider

3. Thevenin Equivalent for Bias Network

Steps:

  1. Vth = Vcc × R2/(R1+R2)
  2. Rth = R1 || R2 = R1×R2/(R1+R2)
  3. IE = (Vth - Vbe)/(Re + Rth/(β+1))
  4. IC ≈ IE for large β

Formula: Vth = Vcc×R2/(R1+R2), Rth = R1||R2

4. Emitter Current Derivation

Steps:

  1. KVL at base: Vth = Ib×Rth + Vbe + Ie×Re
  2. Ie = (β+1)Ib
  3. Substitute: Vth = Ib×Rth + Vbe + (β+1)Ib×Re
  4. Solve: Ib = (Vth-Vbe)/(Rth + (β+1)Re)
  5. Ic = β×Ib ≈ Ie

Formula: Ic = β(Vth-Vbe)/(Rth + (β+1)Re)

5. h-Parameter Conversion Between Configurations

Steps:

  1. hie = hib × (β+1) for CE from CB
  2. hfe = -β for CE from CB
  3. hre = 1 - hfb for CE from CB
  4. hoe = hoe × (β+1) for CE from CB

Formula: hie(CE) = hib(CB) × (β+1)

NUMERICAL PROBLEMS (Solved)

Problem 1: Design CE amplifier bias: VCE=10V, IC=2mA, β=100, VBE=0.7V using voltage divider

Given: R2=10kΩ, Vcc=15V. Find R1, Rc, Re for stability S<10

Solution: Vth = VBE + Ic(Re + Rth/(β+1)) = 0.7 + 2(Re + Rth/101). Choose Re=1kΩ → Vth=2.7+Rth/50.5. Rth=S×Re/(β+1)=10×1k/101=99Ω. R1||R2=99Ω, R2=10k → R1=100Ω. Vth=15×10k/(100+10k)=14.85V. Recalc: 14.85=0.7+2(Re+99/101) → Re=6.58kΩ. VCE=Vcc-Ic(Rc+Re) → Rc=(15-10)/2 - 6.58k = -4.08k (invalid). Redesign: Choose Rc=2kΩ → VCE=15-2(2+6.58)=15-17.16=-2.16V (sat). Choose Ic=1mA: Rc=(15-10)/1 - 6.58= -1.58k. Choose VCE=5V: Rc=(15-5)/1 - 6.58=3.42kΩ

Answer: R1≈100Ω, Rc≈3.4kΩ, Re≈6.6kΩ (adjust for valid VCE)

Problem 2: Calculate stability factor S for fixed bias vs voltage divider bias

Given: β=100, Rb=200kΩ, Re=1kΩ for voltage divider, Rth=20kΩ

Solution: Fixed bias: S=β+1=101. Voltage divider: S=(β+1)/(β+1+Rth/Re)=101/(101+20k/1k)=101/121=0.835

Answer: Fixed: S=101 (poor), Divider: S=0.835 (good)

Problem 3: Find h-parameters from datasheet: β=150, IC=1mA → hie=?

Given: VT=26mV, hoe=20μS

Solution: hie = β×VT/IC = 150×26/1 = 3900Ω = 3.9kΩ. hfe=β=150. hoe=20μS. hre≈0 (neglect)

Answer: hie=3.9kΩ, hfe=150, hoe=20μS
